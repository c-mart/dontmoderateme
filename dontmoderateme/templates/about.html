Don't Moderate Me is a service that lets you monitor any publicly visible URL for presence or absence of any text that you specify.

DMM was created by Chris Martin. See Introducing Don't Moderate Me on Chris Martin's blog for some background and philosophy behind DMM. The full source code can be found on GitHub.
